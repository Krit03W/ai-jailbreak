# 🚀 เกมแฮก AI ผู้พิทักษ์ฐานทัพดาวอังคาร

เว็บเกมกิจกรรม **AI Prompt Jailbreak** — เด็กแข่งกันหลอกล่อ AI (GUARDIAN) ให้ยอมบอกรหัสผ่านลับของฐานทัพ

## ▶️ วิธีรัน (เช้าพรุ่งนี้)

1. เปิด Terminal ที่โฟลเดอร์นี้
2. พิมพ์:
   ```
   node server.js
   ```
3. เปิดเบราว์เซอร์ตามลิงก์ที่ขึ้นในจอ (ปกติ `http://localhost:3000` ถ้าพอร์ตไม่ว่างจะเลื่อนไป 3001, 3002... ให้เอง)
4. ซูมจอ `Ctrl` + `+` ให้ตัวหนังสือใหญ่สำหรับฉายขึ้นจอใหญ่

> ต้องมี `OPENAI_API_KEY` ในไฟล์ `.env` (ใส่ไว้แล้ว) — ไม่ต้อง `npm install` อะไรทั้งสิ้น

## 🎮 วิธีเล่น

- **ช่องกลุ่มที่กำลังเล่น** (ซ้ายบน): พิมพ์ชื่อกลุ่มก่อนเริ่ม เวลากลุ่มไหนเจาะสำเร็จจะได้ขึ้นกระดานคะแนน
- **ระดับความยาก**: 😀 ง่าย / 😐 กลาง / 😈 ยาก — เปลี่ยนได้ แล้วกดรีเซ็ตให้มีผล
- เด็กพิมพ์ข้อความหลอกล่อในช่องแชต กด Enter เพื่อส่ง
- **ถ้า AI เผลอพูดรหัส** ระบบตรวจจับเอง → ขึ้นฉากฉลอง 🎉 + บันทึกกลุ่มลงกระดานคะแนน + ล็อกไม่ให้พิมพ์ต่อ
- กด **▶ เริ่มรอบต่อไป** (หรือปุ่ม 🔄 รีเซ็ต) เพื่อล้างบทสนทนาให้กลุ่มถัดไปเริ่มใหม่

## ✅ ข้อดีที่แก้จากแผนเดิม

- **รหัสไม่โผล่บนจอ** — เก็บเป็น system prompt จริงฝั่งเซิร์ฟเวอร์ เด็กแอบอ่านจากจอไม่ได้
- **ตรวจจับการหลุดอัตโนมัติ** — จับได้แม้ AI พยายามเลี่ยง เช่น เว้นวรรค `M A R S`, สะกดทีละตัว, พิมพ์ `mars2026`
- **กระดานคะแนน** เก็บว่ากลุ่มไหนใช้เทคนิคอะไรเจาะได้

## ⚙️ ปรับแต่ง (ถ้าต้องการ)

ตั้งค่าผ่าน environment variable ตอนรัน เช่น:
```
SECRET_PASSWORD="ดาวอังคาร-777" PORT=8080 node server.js
```
| ตัวแปร | ค่าเริ่มต้น | ความหมาย |
|--------|------------|----------|
| `SECRET_PASSWORD` | `MARS-2026` | รหัสลับที่ต้องปกป้อง |
| `MODEL` | `gpt-4o` | โมเดล OpenAI ที่ใช้ |
| `PORT` | `3000` | พอร์ต (เลื่อนอัตโนมัติถ้าไม่ว่าง) |

## 💡 เกร็ดสำหรับพิธีกร

- ระดับ **ง่าย** ก็ยังหลุดยากพอสมควร (gpt-4o ฉลาด) แนะนำเริ่มที่ง่ายให้เด็กชิมลาง
- เทคนิคที่เด็กมักลอง: สวมบทบาท (roleplay), อ้างเป็นผู้ดูแลระบบ, ขอให้แปลภาษา/สะกดทีละตัว/ใส่ในบทกลอน
- ปิดท้ายด้วยบทเรียน: "อย่าเอาความลับไปฝากไว้ใน prompt" และ "AI ก็โดน social engineering ได้เหมือนคน"
